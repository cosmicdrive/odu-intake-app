# One Dark Universe Intake App

A simple client intake application built to collect user information for guided meditation sessions under the One Dark Universe brand.

## Purpose

This project is designed to:
- capture client information
- practice full-stack development fundamentals
- build a real-world tool that can be expanded into a production system

## Features

- User input form (name, email, message)
- JavaScript form handling
- Real-time data capture via browser console

## Tech Stack

- HTML
- CSS
- JavaScript

## Status

In progress — actively being developed as part of a 90-day build system.

## Next Steps

- Add form validation
- Improve UI/UX
- Connect to a database (Supabase)
- Deploy as a live application
