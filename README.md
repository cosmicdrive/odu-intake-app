# One Dark Universe Intake App

A full-stack intake and workflow management application built for One Dark Universe using JavaScript and Supabase.

## Purpose

This project was created to explore full-stack application development while building a real-world system for collecting, managing, and organizing client intake information.

The application serves as a learning platform for modern software development, database integration, workflow automation, and AI-native systems thinking.

## Current Features

### Intake Management

* Collect client name, email, and intake message
* Form validation for required fields
* Dynamic user feedback and confirmation messages

### Database Integration

* Store submissions in Supabase
* Retrieve and display stored submissions
* Real-time submission rendering

### Workflow Tracking

* Submission status tracking
* Priority classification system
* Timestamp recording and display
* Operational submission metrics

### User Experience

* Responsive form layout
* Dynamic content rendering
* Submission count dashboard
* Visual workflow indicators
